Issue 5614: sourceURL directive not working in Console panel with eval()'ed code

Steps to reproduce

  • Open Firebug
  • Enable the Console and the Script panel and switch to the Console panel
  • Uncheck Show XMLHttpRequests within the Console panel options
  • Reload the page
    ⇒ An error message will be listed with index.html (line 5) as source
  • Check Show XMLHttpRequests within the Console panel options
  • Reload the page again
    ⇒ An error message will be listed with index.html (line 5) as source

Observed

  • The script containing the eval() will be shown as source

Expected

  • The script containing the error (remote.js (line 1)) should be shown as source
