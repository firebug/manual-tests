Issue 4505: Improved auto-completion for CSS functions

Steps to reproduce

  1. Open Firebug
  2. Switch to the HTML panel and there to the Style side panel
  3. Inspect the blue <div> above
  4. Inside the Style side panel double-click within the #element1 stylesheet to add a new CSS property
  5. Type -moz-transform as property name
  6. Press Tab to get to the value editor
  7. Press Down
    ⇒ Auto-completion shows skew(), while the word is selected and the braces shown in gray (as hint, that skew is a function) (not implemented yet)
  8. Press Right, Tab or Enter
    ⇒ Auto-completion shows skew() (in black) and puts the text cursor between the braces (not implemented yet)

Expected Result

  • The Auto-completion should work similar to the one used for the Command Line by showing the braces as completion suggestion
  • When the user accepts the suggestion, the text cursor should be put between the braces
