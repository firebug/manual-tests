Issue 6579: Highlight used fonts asynchronously

Text using a web font

Steps to reproduce

  1. Open Firebug
  2. Switch to the HTML panel and there to the Style side panel
  3. Reload the page via Ctrl + F5

Expected result

  • The display of the Style panel contents should not wait until the font "TitilliumMaps" is loaded and display "Courier New" as the used font.
  • As soon as "TitilliumMaps" is loaded, the font should be highlighted inside the Style panel.
