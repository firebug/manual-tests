Issue 2852: Hovering JavaScript variables while debugging is not always showing an info popup

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Script panel
  3. Reload the page
    ⇒ The script execution should stop at the debugger keyword
  4. Hover over the different variables and properties inside the Script panel

Observed result

  • There's no infotip shown for the variables except for the global ones and name.
  • name shows an empty string

Expected result

  • Hovering arrayInObject should show the array like hovering array in line 10
  • Hovering the index should show the value of the array at that index (271)
  • Hovering project should show the object with name an version
  • Hovering name should show "Firebug"
  • Hovering version should show 1.6
