Issue 2183: Editing a text node with "Show Full Text" off displays cropped text instead of full text

Steps to reproduce

  1. Open Firebug, select the HTML panel
  2. Inspect the text in the green box
  3. Make sure "Show Full Text" options is off
  4. Click the text in the HTML panel to show up an inline editor
  5. The inline editor should display full text (not using ellipses)
