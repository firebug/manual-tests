Issue 7262: HTML panel allows to inspect elements not added to the page

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Script panel
  3. Reload the page
    ⇒ The script should stop at line 19 (debugger;)
  4. Switch to the Watch side panel
    ⇒ The panel should list a div property with a semi-transparent (because it's not added to the page yet) div#element as value.
  5. Click the div#element value

Observed result

  • Firebug switches to the HTML panel and displays the element and by that hides the page's HTML structure.

Expected result

  • Firebug should not allow to inspect the element within the HTML panel.
