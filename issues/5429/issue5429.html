Issue 5429: Show @document rules inside CSS panel

Steps to reproduce

  1. Open Firebug
  2. Switch to the CSS panel
  3. Select issue5429.html from the Style Location Menu

Observed result

  • An info message is shown inside the CSS panel that there are no CSS rules.

Expected result

  • The following should be displayed inside the CSS panel:
    @-moz-document regexp("^.*issue5429.html$") { #element1 { width: 100px; height: 100px; background: #8C8CFF -moz-linear-gradient(135deg, #788CFF, #B4C8FF); } }
