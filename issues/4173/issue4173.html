Issue 4173: Window/Document menu

Console panel

  1. Open Firebug
  2. Enable and switch to the Console panel
  3. Open the Frame Menu (not existing yet)
    There should be three entries listed. One representing the top window, one for frame 1 and one for frame 2.
  4. Choose Frame 1 (issue4173-frame1.html) from the menu
  5. Type document.body.textContent.trim() into the Command Line and hit Enter

  • The console should output "frame 1"
