This test-case is intended for testing various breakOnError feature
Jan Odvarko, odvarko@gmail.com

Break On Next Error

  1. Open Firebug and enable the Console and Script panel
  2. Switch to the Console panel and click Break On All Errors.
  3. Press this button
  4. Firebug should switch to the Script panel and break in the debugger at the line where the error happened.

Error Breakpoint

  1. Open Firebug and enable the Console and Script panel
  2. Press this button
  3. You should see an error log in the Console panel allowing to set a breakpoint on it
  4. Press this button (the same error)
  5. Firebug should switch to the Script panel and break in the debugger at the line where the error happened.

Break On Exceptions

  1. Open Firebug and enable the Script panel
  2. Switch to the Script panel and check Break On Exceptions option in the panel tab options menu.
  3. Press this button
  4. Firebug should switch to the Script panel and break in the debugger.

Ignore Caught Exceptions

  1. Open Firebug and enable the Script panel
  2. Switch to the Script panel and check Break On Exceptions and Ignore Caught Exceptions option in the panel tab options menu.
  3. Press this button
  4. Firebug should switch to the Script panel and break in the debugger.
