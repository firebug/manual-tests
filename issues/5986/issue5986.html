Issue 5986: Allow the Script panel to deal with web workers

Steps to reproduce

  1. Open Firebug
  2. Enable and switch to the Script panel
  3. Switch to the Workers side panel (not existing yet)
  4. Check the Workers side panel option Break On Start (not existing yet)
  5. Reload the page

Expected result

  • The Script panel should display the contents of the web worker worker.js
  • The Workers side panel should list worker.js
  • The script execution should stop at the first line of worker.js
